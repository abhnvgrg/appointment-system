# Appointment Reminder System

A full-stack serverless web application for booking appointments, receiving instant confirmations, and getting automated reminders.

##  Architecture

### Frontend
- **React 18** with Vite for fast development
- Appointment booking form with validation
- Dashboard displaying all appointments
- Real-time updates and user feedback

### Backend
- **Supabase Edge Functions** - Serverless endpoints for:
  - `book-appointment`: Handle appointment bookings and send confirmations
  - `get-appointments`: Fetch all appointments
  - `check-reminders`: Scheduled function to send reminders 1 hour before appointment

### Database
- **PostgreSQL** (via Supabase)
- `appointments` table with fields:
  - `id`: Unique identifier
  - `name`: User's name
  - `phone`: Phone number for SMS/WhatsApp
  - `time`: Scheduled appointment time
  - `reminder_sent`: Flag to prevent duplicate reminders
  - `created_at`: Booking timestamp

### Integrations
- **Twilio** - SMS and WhatsApp message delivery

##  Quick Start

### Prerequisites
- Node.js 18+
- Supabase account (https://supabase.com)
- Twilio account (https://www.twilio.com) - Optional but recommended

### 1. Setup Frontend

```bash
# Install dependencies
npm install

# Create .env.local file
cp .env.example .env.local

# Update with your Supabase credentials
# VITE_SUPABASE_URL=https://your-project.supabase.co
# VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 2. Setup Supabase

#### Create Database Table
In your Supabase dashboard, run this SQL:

```sql
CREATE TABLE appointments (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name VARCHAR(255) NOT NULL,
  phone VARCHAR(20) NOT NULL,
  time TIMESTAMPTZ NOT NULL,
  reminder_sent BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_appointments_time ON appointments(time);
CREATE INDEX idx_appointments_reminder_sent ON appointments(reminder_sent);
```

#### Deploy Edge Functions

```bash
# Install Supabase CLI
npm install -g @supabase/cli

# Link to your Supabase project
supabase link --project-ref your-project-id

# Deploy functions
supabase functions deploy book-appointment
supabase functions deploy get-appointments
supabase functions deploy check-reminders

# Set environment variables in Supabase dashboard
# TWILIO_ACCOUNT_SID=your-account-sid
# TWILIO_AUTH_TOKEN=your-auth-token
# TWILIO_PHONE_NUMBER=+1234567890
# TWILIO_WHATSAPP_FROM=whatsapp:+14155238886
```

### 3. Setup Twilio (Optional)

1. Get your Twilio credentials from https://console.twilio.com
2. Set environment variables in Supabase:
   - `TWILIO_ACCOUNT_SID`
   - `TWILIO_AUTH_TOKEN`
   - `TWILIO_PHONE_NUMBER` (your Twilio phone number for SMS confirmations)
   - `TWILIO_WHATSAPP_FROM` (your Twilio WhatsApp sender, for example `whatsapp:+14155238886` for the sandbox)
3. If using the Twilio WhatsApp sandbox, have each recipient join your sandbox from their WhatsApp app before reminders are sent.

### 4. Schedule Reminder Checker

In Supabase Edge Functions settings, create a cron job to run `check-reminders` every minute:

```
* * * * * (every minute)
```

Or every 5 minutes for less frequent checks:

```
*/5 * * * * (every 5 minutes)
```

- API requests include Supabase authorization header

- Phone numbers are validated before processing
- CORS headers configured for secure cross-origin requests
